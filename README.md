# Expense Tracker 💰

A lightweight, web-based financial management tool built with **Python** and **Flask**. This application allows users to log daily expenses, categorize spending, and visualize their budget.



## 🚀 Features
* **Add/Edit/Delete Expenses:** Manage your transactions with a simple CRUD interface.
* **Categorization:** Sort spending into categories like Food, Rent, and Entertainment.
* **Data Persistence:** Uses SQLite to store your data locally.
* **Responsive Design:** Clean UI built with CSS and HTML.

## 🛠️ Tech Stack
* **Backend:** Python 3.x, Flask
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** SQLite (via SQLAlchemy)
